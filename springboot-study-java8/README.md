# 跟上 Java 8 教程

这个仓库存储了『跟上Java 8』视频课程源码。

Java8 改变了我们思考和编码的方式，在这里你可以学习到 Java8 中的核心特性以及使用场景，跟我学习 Java8，让你的编程更轻松！

课程的讲授方式原则是：**What**、**Why**、**How**

我们会说清楚每个技能是什么，为什么是这样，以及编码演示他们的使用方法以及场景，
最后会讲解 Java 8 中的最佳实践以及正确的函数式编程姿势。
